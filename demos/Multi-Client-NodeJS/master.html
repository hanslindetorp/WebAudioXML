Client 0 - touch 0 (Drum Loop):

relY - lowpass Filter Cutoff, relX - lowpass Filter Q, deviceOrientation.gamma - Reverb send level

Client 1 - touch 0 (Bass Sawtooth Synth):

relX - frequency (MIDI notes) on oscillator, deviceOrientation.gamma - detune on lowpass filter cutoff, touch start/end - trig envelopes (filter + gain)

Client 2 - touch 0 (Lead Square Synth):

relX - frequency (MIDI notes) on oscillator, force - detune on lowpass filter cutoff, relY - reverb send, touch start/end - trig envelopes (filter + gain)
